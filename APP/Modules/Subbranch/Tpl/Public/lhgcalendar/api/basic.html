lhgcalendar运行环境

lhgcalendar是基于浏览器的HTML、CSS和Javascript环境开发的，因此原则上市场上绝大部分浏览器我们都能完美的运行，目前已测试的浏览器兼容列表如下：IE 6.0+,Firefox 3.0+,Opera 9.6+,Chrome 1.0+,Safari 3.22+ (+号代表往后所有新版本)。lhgcalendar是完全基于浏览器环境的，因此不存在操作系统选择性，是完全跨平台的。lhgcalendar同时也可以应用在任何的服务端语言环境下，例如：PHP、ASP、ASP.NET、JAVA等，能够非常灵活简单的和您的系统实现完美的无缝衔接。

lhgcalendar基本使用指南

您只需按照以下简单步骤即可学会使用lhgcalendar：

  1. 下载最新版本lhgcalendar：http://lhgcore.com/lhgcalendar.rar
  2. 将下载的压缩文件解压缩，上传其中的lhgcore.min.js以及lhgcalendar.min.js，lhgcalendar.css和images到网站相应文件夹中
  3. 在需要调用lhgcalendar弹出日历的网页head标签结束之前添加以下代码： 
    <script type="text/javascript" src="lhgcore.min.js"></script>
    <script type="text/javascript" src="lhgcalendar.min.js"></script>
    注：如果将窗口组件做为jQ的组件来使用就不需要加载lhgcore.min.js库文件，而要加载jQ库文件。 
    <script type="text/javascript" src="jquery-1.6.min.js"></script>
    <script type="text/javascript" src="lhgcalendar.min.js"></script>
    注：作为jQ组件来使用一定要注意把lhgdialog.min.js文件中代码最后小括号里的lhgcore改成jQuery，为里要区分大小写
    2.0.2以后的版本不再需要改这里了，加载jQuery的库后直接就可以使用。
  4. lhgcalendar提供两种方式弹出窗口： 
    <head>
<script type="text/javascript">
    //第一种jQ调用方式
J(function(){
    J('#dg').calendar({ 参数 });
});

//如果作为jQ组件来用的话J就得换成$
$(function(){
    $('#dg').calendar({ 参数 });
});

//第二种普通函数式调用
function opcal( id )
{
    J.calendar.Show({ 参数 });
	
    //如果作为jQ组件来用的话J就得换成$
    $.calendar.Show({ 参数 });
}
</script>
</head>
<body>
    <input type="text" id="dg"/>
    <input type="text" value="opendg2" onclick="opcal();"/>
</body>
    注：第一种方式适合于简单的调用弹出日历的方式，但必须放到J(function(){ ... });里
           第二种方式适合于传参数的弹出日历。

经过以上步骤，你就已经成功的将lhgcalendar弹出日历组件安装在你的网页中。若需要学习更多的使用技术，请参阅：API参数控制接口。
