初始化参数列表

初始化参数列表：即为J.calendar.Show({ 这里的参数 })或J('#id').calendar({ 这里的参数 });

  1. id：输入框的id

    参数值：输入框的id。使用其他的元素如:<img><div>等来调用弹出日期框时一定要指定输入日期的文本框的id。

  2. format：自定义日期格式

    参数值：默认'yyyy-MM-dd'。yMdHms分别代表年月日时分秒，你可以任意组合这些元素来自定义你个性化的日期格式。请见下表：

    日期格式表

    格式 说明
    yy 将年份表示为二位数字。
    yyyy 将年份表示为四位数字。如果少于四位数，前面补零。
    M 将月份表示为从 1 至 12 的数字
    MM 同上，如果小于两位数，前面补零。
    d 将月中日期表示为从 1 至 31 的数字。
    dd 同上，如果小于两位数，前面补零。
    H 将小时表示为从 0 至 23 的数字。
    HH 同上，如果小于两位数，前面补零。
    m 将分钟表示为从 0 至 59 的数字。
    mm 同上，如果小于两位数，前面补零。
    s 将秒表示为从 0 至 59 的数字。
    ss 同上，如果小于两位数，前面补零。

    示例(只列出了常用的几种格式，如需要其它格式请自行设置)

    格式字符串 值
    yyyy-MM-dd HH:mm:ss 2008-03-12 19:20:00
    yy年M月 08年3月
    yyyyMMdd 20080312
    MM/dd/yyyy 03/12/2008

    动态变量表

    格式 说明
    %y 当前年
    %M 当前月
    %d 当前日
    %H 当前时
    %m 当前分
    %s 当前秒

  3. btnBar：是否显示下面的按钮栏

    参数值：默认true(显示），false(不显示)。

  4. minDate：最小日期

    参数值：默认null，注意格式要为 yyyy-MM-dd。

  5. maxDate：最大日期

    参数值：默认null，注意格式要为 yyyy-MM-dd。

  6. targetFormat：目标文本框的日期格式

    参数值：null，当有2个文本框之间进行日期范围限制时指来指定对应的文本框的日期格式。

  7. disWeek：无效周

    参数值：null，可以使用此功能禁用周日至周六所对应的日期。

  8. disDate：无效日期

    参数值：null，可以使用此功能禁用所指定的一个或多个日期。

  9. enDate：有效日期 

    参数值：默认false，为true时,无效日期变成有效日期，一般和disDate配合使用。

  10. real：存储真实日期值的控件的ID

    参数值：null，指定一个控件的ID,必须具有value属性(如input),用于存储真实值。

  11. onSetDate：点击某一日期时执行的函数

    当选择一个日期时点击这个日期时就会触发这个函数。

  12. linkageObj：联动弹出日历的目标对象(2.0.3新增)

    如果想要2个文本框实现日期间的联动，要加此参数，此参数指定的是第2个自己弹出的文本框的对象，这里要注意第二个文本框要使用onfocus来弹出日历。

其中一些参数的用法请参照示例中的使用方法

API函数接口列表

API接口列表：

  1. Show() ：显示日历

    无参数，jQ调用方式不需要加此方法。

  2. hideCalendar() ：隐藏日历

    无参数。

  3. getDateStr(param) ：获取指定格式的日期

    参数1：指定的日期格式 (y:返回年，M:返回月，d:返回日，H:返回小时，m:返回分钟，s:返回秒，date:返回年月日，格式yyyy-MM-dd，dateTime:返回年月日时分秒，格式yyyy-MM-dd HH:mm:ss)

  4. formatDate(data,format) ：格式化日期

    参数1：要格式化的日期，参数2：指定的格式，如：'MM/dd/yyyy'。

API属性接口列表：

  1. inpObj ：文框架对象

    显示日期的文本框。

  2. cal ：日历的DOM对象

    通过此对象可以对日历DOM对象进行操作。
